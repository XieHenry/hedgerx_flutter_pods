# HedgerXFlutter Pods

This repository contains the CocoaPods spec for HedgerX Flutter Engine (xcframework)
distributed via Alibaba Cloud OSS.

## Usage

Add this to your iOS project's Podfile:

```ruby
pod 'HedgerXFlutter', :git => 'https://github.com/your-company/hedgerx_flutter_pods.git', :tag => '1.0.1'
